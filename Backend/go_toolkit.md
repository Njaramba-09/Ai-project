# **Beginner’s Toolkit for Go (Golang)**

**Moringa AI Capstone Project**

---

## **1. Title & Objective**

**Title:** Beginner’s Toolkit for Go (Golang)

**Objective:**
This toolkit introduces beginners to the Go programming language and demonstrates how to use Generative AI to learn Go effectively. The project walks through installation, creating a simple web service to serve a frontend, troubleshooting issues, and documenting the learning journey.

By the end, learners should feel confident running a Go backend and connecting it with a simple HTML/CSS frontend.

---

## **2. Quick Summary of the Technology**

**What is Go?**
Go (Golang) is an open-source programming language created by Google. It is fast, simple, and excellent for backend development.

**Where Go Is Used:**

* Web servers & REST APIs
* Cloud and distributed systems
* Backend services and microservices
* CLI tools

**Real-World Use Case:**
Uber uses Go for high-performance backend services.

---

## **3. System Requirements**

You will need:

* Windows, macOS, or Linux
* Go installed from [https://go.dev/dl](https://go.dev/dl)
* VS Code
* Terminal (PowerShell, Command Prompt, or Linux/macOS Terminal)
* A web browser

---

## **4. Installation & Setup Instructions**

### **Install Go on Ubuntu (Linux)**

```
sudo apt update
sudo apt install golang-go -y
```

**Verify installation:**

```
go version
```

**Optional: Install latest Go manually**
